# Secret List Default Schema

This file is the built-in fallback schema shipped inside the SecretList package.
It is read-only at runtime (packaged apps cannot write to their own install folder).
To customize your schema, use the "Export Schema" menu item in the app to
save a copy somewhere writable, edit that copy, then use "Import Schema" to load
it as your active schema.md.
